ExpenseCounter

ExpenseCounter is an Android application developed in Java focused on helping users track and understand their everyday expenses in a simple and practical way.

The project starts with basic expense tracking and is planned to grow into a more complete personal finance tool, incorporating real-life budgeting and saving techniques.
